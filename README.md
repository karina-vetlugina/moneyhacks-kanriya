# Money Hacks by Kanriya

**ElleHacks 2026**

---

## Overview

Money Hacks is a mobile-first interactive simulation game designed to teach youth practical financial skills through real-life decision making.

Players progress through common financial milestones such as earning paychecks, saving toward major goals, budgeting expenses, building credit, and learning basic investing concepts. The game emphasizes long-term thinking by showing how everyday choices impact financial outcomes over time.

---

## Key Features

- Interactive, choice-based financial simulation
- Realistic scenarios inspired by early adulthood
- Savings goals and progress tracking
- Credit score introduction and education
- Financial tips unlocked through gameplay
- Mobile-first design with narrative-driven flow

---

## Tech Stack

- HTML, CSS, JavaScript
- Mobile-first web architecture
- Cursor used for rapid prototyping and logic implementation
